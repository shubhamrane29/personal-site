# Portfolio Redesign using Flask

This project is a redesign of my personal portfolio website, implemented using Flask, a Python web framework. The website hosts a form that is connected to a SQL database through Flask. The website is currently hosted on Render and can be accessed at [www.shubhamrane.com](http://www.shubhamrane.com).

## Project Overview

The goal of this project was to revamp my portfolio website to showcase my skills and projects in a more organized and visually appealing manner. By using Flask, I aimed to create a dynamic and interactive user experience while also learning more about web development and backend technologies.

## Features

- **Portfolio Display**: Showcases my projects, skills, and experiences.
- **Contact Form**: Allows visitors to reach out to me directly through the website.
- **SQL Integration**: Stores form submissions in a SQL database.

## Technologies Used

- **Flask**: Python web framework used for backend development.
- **SQLAlchemy**: SQL toolkit and Object-Relational Mapping (ORM) library for Python used for database management.
- **HTML/CSS**: Frontend markup and styling for the website.
- **Render**: Hosting platform used for deploying the Flask application.
- **porkbun**: Domain registrar used for managing the website domain.
- **Free SQL Database**: Hosting service for the SQL server used to store form submissions.
